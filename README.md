# Verilog

## Installation

```sh
brew install icarus-verilog
```

## Compilation

```sh
iverilog -o [output] [file1], [file2], ...
```

## Execution

```sh
$ vvp [file]
```

Since it's already executable, or simply

```sh
$ ./[file]
```
